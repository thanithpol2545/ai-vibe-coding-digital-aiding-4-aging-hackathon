# Self-Recording Protocol — คู่มือการบันทึกวิดีโอด้วยตนเอง

> สำหรับผู้สูงอายุและผู้ดูแลที่ต้องการบันทึกวิดีโอการเคลื่อนไหวมือเพื่อวิเคราะห์ด้วย AI
> อ้างอิงจาก TAS Test, PLOS One Parkinson's screening, MDPI Hand Rehabilitation Platform

---

## 1. อุปกรณ์และสภาพแวดล้อม (Environment & Equipment)

### 1.1 กล้อง (Camera)
| ข้อกำหนด | รายละเอียด |
|----------|-----------|
| ระยะห่าง | **50 ซม. — 1 ม.** จากกล้องถึงมือ |
| มุมกล้อง | **หันหน้าตรงเข้าหากล้อง** (45° - 90° ได้ผลดีที่สุด) |
| ความละเอียด | ≥ 720p (HD) |
| Frame Rate | ≥ 30 FPS |
| การตั้งกล้อง | วางบนโต๊ะ หรือใช้ขาตั้ง ไม่ต้องถือกล้องเอง |

### 1.2 แสง (Lighting)
- ✅ **แสงสว่างเพียงพอ** — ห้องที่มีแสงธรรมชาติหรือแสงไฟสว่าง
- ❌ **หลีกเลี่ยงแสงย้อน (Backlight)** — เช่น ยืนหน้าหน้าต่างตอนกลางวัน
- ❌ **หลีกเลี่ยงเงาตกกระทบบนมือ**
- 💡 เปิดไฟ overhead หรือไฟตั้งโต๊ะส่องมาที่มือ

### 1.3 พื้นหลัง (Background)
- ✅ พื้นหลังสีพื้นเรียบ (ผนังสีขาว/สีอ่อน)
- ❌ ไม่มีวัตถุเคลื่อนไหวหรือคนเดินผ่านด้านหลัง
- ❌ ไม่มีกระจกหรือพื้นผิวสะท้อนแสง

### 1.4 การแต่งกาย (Clothing)
- ✅ เสื้อแขนสั้น หรือพับแขนเสื้อขึ้นเหนือข้อศอก
- ✅ ไม่สวมถุงมือ, แหวน, นาฬิกาข้อมือ, สายรัดข้อมือ
- ❌ หลีกเลี่ยงเสื้อแขนยาวที่บังข้อมือ

### 1.5 อุปกรณ์เสริม
- โต๊ะหรือพื้นผิวเรียบสำหรับวางมือ (สำหรับ Finger Tapping)
- เก้าอี้ที่มีพนักพิง (สำหรับนั่ง)
- ขาตั้งกล้องหรือกองหนังสือสำหรับวางโทรศัพท์

## 2. ท่านั่ง/ท่ายืน (Body Positioning)

### 2.1 การนั่ง (Sitting) — แนะนำสำหรับผู้สูงอายุ
```
🪑 นั่งบนเก้าอี้ที่มีพนักพิง
📏 ระยะห่างจากกล้อง 50 ซม. — 1 ม.
📐 ไหล่ตรง ไม่ก้ม ไม่เอียง
🤲 มืออยู่ในระดับอก (shoulder height)
🖐️ ฝ่ามือหันเข้าหากล้อง
```

- วางเท้าราบกับพื้น
- หลังตรง พิงพนักเก้าอี้
- ไหล่ผ่อนคลาย ไม่เกร็ง
- วางแขนบนโต๊ะถ้ามี (สำหรับ Finger Tapping)

### 2.2 การยืน (Standing) — สำหรับ Reach-to-Target / Arm Raise
```
🧍 ยืนตรง เท้ากว้างเท่าช่วงไหล่
📏 ห่างจากกล้อง 1 ม. (เห็นเต็มตัวท่อนบน)
🤲 มือทั้งสองข้างอยู่ข้างลำตัว (ท่าเริ่มต้น)
🔄 ขยับแขนเข้าหากล้อง (Reach)
```

- สำหรับผู้ที่ยืนได้มั่นคงโดยไม่ต้องเกาะ
- ใช้ Walker หรือเก้าอี้ช่วยทรงตัวได้ถ้าจำเป็น

## 3. การวางมือในเฟม (Hand Placement in Frame)

### 3.1 กฎพื้นฐาน
```
┌─────────────────────────┐
│                         │
│    📹 กล้อง/webcam      │
│                         │
│   ┌─────────────────┐   │
│   │                 │   │
│   │  🤚      🤚    │   │  ← มือทั้งสองข้างอยู่ในเฟรม
│   │  ซ้าย   ขวา    │   │
│   │                 │   │
│   └─────────────────┘   │
│                         │
│       🪑/🧍            │
└─────────────────────────┘
```

### 3.2 ข้อควรระวัง
- ❌ มือไม่อยู่นอกเฟรมกล้อง (Malposition = สาเหตุหลักที่ข้อมูลใช้ไม่ได้)
- ❌ มือบังกัน (Hand overlapping)
- ❌ หันข้างมือ (Side view) — **ต้องหันฝ่ามือเข้าหากล้อง**
- ✅ ตรวจสอบตัวเองในหน้าจอก่อนเริ่มอัด

## 4. แบบทดสอบการเคลื่อนไหว (Exercise Protocols)

### 4.1 Finger Tapping Test — แตะนิ้วชี้-นิ้วโป้ง

```
📋 คำสั่ง:
  "แตะปลายนิ้วชี้กับนิ้วโป้งให้ใหญ่และเร็วที่สุดเท่าที่จะทำได้"

⏱ ระยะเวลา: 10-30 วินาทีต่อมือ
🔄 ทำสลับ: มือขวาก่อน → มือซ้าย (หรือกลับกัน)

📍 ตำแหน่งมือ:
  - ยกมือขึ้นระดับอก
  - ฝ่ามือหันเข้าหากล้อง
  - ปลายนิ้วชี้กับนิ้วโป้งห่างกัน ≈ 5 ซม. ก่อนเริ่ม

🗣 คำแนะนำเสียง (text-to-speech):
  "พร้อม... 3... 2... 1... เริ่ม!"
  "แตะให้ใหญ่ เร็ว และสม่ำเสมอ"
  "5 วินาทีสุดท้าย... 3... 2... 1... หยุด!"

📊 สิ่งที่ AI วัด:
  - จำนวนครั้งที่แตะ (Tap Count)
  - ความเร็วในการแตะ (Tapping Speed)
  - ความสม่ำเสมอ (Tap Regularity)
  - ความกว้างของการแตะ (Amplitude)
```

### 4.2 Hand Opening-Closing — กํามือ-แบมือ

```
📋 คำสั่ง:
  "กํามือให้แน่นที่สุด แล้วแบมือออกกว้างที่สุด สลับกันเร็วที่สุด"

⏱ ระยะเวลา: 10-15 วินาที
🤲 ทำทั้งสองมือพร้อมกัน หรือทีละข้าง

📍 ตำแหน่งมือ:
  - ยกมือขึ้นระดับอก
  - ฝ่ามือหันเข้าหากล้อง
  - กํามือ → เหยียดนิ้วทุกนิ้วออก

📊 สิ่งที่ AI วัด:
  - Range of Motion (ROM) ของนิ้ว
  - ความเร็วในการเปิด-ปิด
  - Finger individuation
```

### 4.3 Reach-to-Target — เอื้อมแขนไปข้างหน้า

```
📋 คำสั่ง:
  "เอื้อมแขนทั้งสองข้างไปข้างหน้าให้สุด แล้วดึงกลับ"

⏱ ระยะเวลา: 10-15 วินาที (5-7 ครั้ง)
🧍 ท่ายืน หรือนั่งก็ได้

📍 ตำแหน่ง:
  - เริ่ม: แขนแนบลำตัว งอศอก 90°
  - เอื้อม: เหยียดแขนตรงไปข้างหน้า
  - กลับ: ดึงแขนกลับมาที่เดิม

📊 สิ่งที่ AI วัด:
  - Reach Time (เวลาเอื้อมถึงเป้าหมาย)
  - Path Efficiency (เส้นทางตรง vs จริง)
  - Smoothness (SPARC/Jerk)
```

### 4.4 Arm Raise — ยกแขนขึ้น-ลง

```
📋 คำสั่ง:
  "ยกแขนทั้งสองข้างขึ้นเหนือศีรษะ แล้วลดลงข้างลำตัว"

⏱ ระยะเวลา: 10-15 วินาที
🧍 ท่ายืน (ถ้าเป็นไปได้)

📍 ตำแหน่ง:
  - เริ่ม: แขนข้างลำตัว
  - ยก: เหยียดแขนขึ้นเหนือหัว
  - กลับ: ลดแขนลงข้างลำตัวช้าๆ

📊 สิ่งที่ AI วัด:
  - Shoulder flexion angle
  - ความเร็วในการยก
  - Tremor ระหว่างยก
```

## 5. ขั้นตอนการบันทึก (Recording Workflow)

### Step-by-Step สำหรับผู้ใช้

```
📱 STEP 1: เตรียมอุปกรณ์
  ✅ เปิดคอมพิวเตอร์/โทรศัพท์
  ✅ เปิดไฟในห้องให้สว่าง
  ✅ นั่งบนเก้าอี้ที่มีพนักพิง
  ✅ ถอดนาฬิกา แหวน ถุงมือ

📱 STEP 2: เปิดแอปพลิเคชัน
  ✅ เปิดเว็บเบราว์เซอร์
  ✅ เข้าไปที่เว็บแอป
  ✅ อนุญาตให้ใช้กล้อง (Allow Camera)

📱 STEP 3: จัดตำแหน่ง
  ✅ นั่งห่างจากกล้อง ≈ 50 ซม.
  ✅ ยกมือขึ้นระดับอก
  ✅ ฝ่ามือหันเข้าหากล้อง
  ✅ ตรวจสอบในหน้าจอว่ามืออยู่ในเฟรม

📱 STEP 4: เลือกแบบทดสอบ
  ✅ Finger Tapping (แตะนิ้ว)
  ✅ Hand Open-Close (กํามือ-แบมือ)
  ✅ Reach-to-Target (เอื้อมแขน)

📱 STEP 5: เริ่มบันทึก
  ✅ กดปุ่ม "Start Recording"
  ✅ รอสัญญาณนับถอยหลัง 3-2-1
  ✅ ทำตามคำแนะนำบนหน้าจอ
  ✅ หยุดเมื่อได้ยินสัญญาณ

📱 STEP 6: ดูผลลัพธ์
  ✅ รอ AI วิเคราะห์
  ✅ ดูผลการประเมิน
  ✅ บันทึกหรือแชร์ผลได้
```

## 6. ข้อผิดพลาดที่พบบ่อย (Common Mistakes)

| ปัญหา | สาเหตุ | วิธีแก้ |
|-------|--------|--------|
| มือหลุดเฟรม | วางมือไกลจากกล้องเกินไป | ขยับมือเข้าใกล้กล้อง หรือขยับกล้องออก |
| ตรวจจับไม่เจอ | แสงไม่พอ / มือหันข้าง | เปิดไฟเพิ่ม / หันฝ่ามือเข้ากล้อง |
| Data น้อยเกินไป | ขยับมือช้า/น้อยเกินไป | กระตุ้นให้ทำเร็วขึ้น |
| มือบังกัน | วางมือซ้อนกัน | กางมือทั้งสองข้างให้ห่างกัน |
| Detection ต่ำ | กล้องความละเอียดต่ำ | ใช้กล้อง 720p ขึ้นไป |
