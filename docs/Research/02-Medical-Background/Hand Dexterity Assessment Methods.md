# Hand Dexterity Assessment Methods — วิธีการประเมินความคล่องมือ

> **มาตรฐานทางคลินิกสำหรับประเมินความคล่องแคล่วของมือ**
> ที่สามารถนำมาประยุกต์ใช้กับการวิเคราะห์ผ่านวิดีโอ

---

## 1. Purdue Pegboard Test (PPBT)

| หัวข้อ | รายละเอียด |
|-------|-----------|
| **วัด** | Gross hand/wrist movement + fine finger dexterity |
| **อายุ** | 6+ ปี, ผู้ใหญ่, ผู้สูงอายุ |
| **เวลา** | < 5 นาที |
| **อุปกรณ์** | Pegboard, pins, washers, collars, timer |

### วิธีการทดสอบ
- **Part 1 — Picking Up and Placing:** ใช้มือเดียวหยิบ pins ใส่รูให้ได้มากที่สุดใน 30 วินาที (ทำทั้งขวา ซ้าย และสองมือ)
- **Part 2 — Assembly:** ใช้สองมือประกอบ pin-washer-collar-washer ให้มากที่สุดใน 30 วินาที

### ข้อควรรู้
- ค่าใช้จ่าย: ~$125-200
- ใช้ประเมิน cursory movements ของมือและข้อมือ
- คะแนน = จำนวนชิ้นที่ใส่ได้

## 2. Box and Block Test (BBT)

| หัวข้อ | รายละเอียด |
|-------|-----------|
| **วัด** | Gross manual dexterity |
| **อายุ** | 6+ ปี, ผู้ใหญ่, ผู้สูงอายุ |
| **เวลา** | 2-5 นาที |
| **อุปกรณ์** | กล่อง 2 ช่อง, 150 blocks |

### วิธีการทดสอบ
- นั่งตรงข้ามผู้ทดสอบ
- ย้าย blocks จากช่องหนึ่งไปอีกช่องหนึ่ง
- นับจำนวนที่ย้ายได้ใน 60 วินาที
- วัดทีละมือ

### ข้อควรรู้
- ค่าใช้จ่าย: ~$200

## 3. Nine-Hole Peg Test (9HPT / NHPT) ⭐

| หัวข้อ | รายละเอียด |
|-------|-----------|
| **วัด** | Fine motor dexterity |
| **อายุ** | 18+ |
| **เวลา** | 1-3 นาที |
| **อุปกรณ์** | 9 pegs, pegboard, stopwatch |

### วิธีการทดสอบ
1. ใช้มือเดียว หยิบ peg แต่ละอันจาก container
2. ใส่ลงใน 9 รูบน board
3. ดึงออกกลับมาใส่ container
4. **จับเวลา** — นาฬิกาจับเวลาไม่หยุดแม้ peg หล่น
5. ทำทีละมือ

### Normative Data
- เวลาที่เร็วขึ้น = dexterity ดีขึ้น
- **Hand dexterity** เป็น independent predictor ของ cognitive performance
- สัมพันธ์กับ verbal fluency, memory, executive speed, inhibition

### ทำไม 9HPT ถึงสำคัญกับ Hackathon
| เหตุผล | คำอธิบาย |
|--------|---------|
| **Simple setup** | แค่ peg + board → video analysis ทำได้ง่าย |
| **Timed + error-tolerant** | จับเวลาต่อเนื่องแม้ทำผิด → เหมือน real-world |
| **Fine motor sensitive** | ไวต่อความแตกต่างของ dexterity |
| **L vs R comparison** | เปรียบเทียบสองข้างได้ดี |

## 4. Jebson-Taylor Hand Function Test (JHFT)

| หัวข้อ | รายละเอียด |
|-------|-----------|
| **วัด** | ADL-related hand function |
| **อายุ** | 6+ ปี, ผู้ใหญ่, ผู้สูงอายุ |
| **เวลา** | ~15 นาที |
| **อุปกรณ์** | 7 subtests: writing, cards, small objects, feeding, stacking, etc. |

### 7 Subtests
1. Writing (เขียน)
2. Turning cards (พลิกไพ่)
3. Picking up small objects (หยิบของเล็ก)
4. Simulated feeding (ตักอาหาร)
5. Stacking checkers (ซ้อนหมาก)
6. Moving large light objects (ย้ายของเบา)
7. Moving large heavy objects (ย้ายของหนัก)

### ข้อควรรู้
- ค่าใช้จ่าย: ~$300+
- อิงกับ ADLs มากที่สุด
- แต่ใช้เวลานานและอุปกรณ์เยอะ

## 5. Functional Dexterity Test (FDT)

| หัวข้อ | รายละเอียด |
|-------|-----------|
| **วัด** | Dynamic dexterity, in-hand manipulation, grasping patterns |
| **อายุ** | 6+ ปี, ผู้ใหญ่, ผู้สูงอายุ |
| **เวลา** | 2-5 นาที |
| **อุปกรณ์** | Pegboard, pegs (ใหญ่กว่า 9HPT) |

### วิธีการทดสอบ
- Flip pegs ในรูปแบบ zig-zag
- มีเวลา 55 วินาที
- **Penalty:** +เวลาเมื่อ peg หล่น, หมุนแขนผิดท่า, แตะ board

### Scoring
```
คะแนน = เวลาที่ใช้ (วินาที) + penalty points
```
- Penalty points: dropping peg (+5s), supination (+5s), touching board (+5s)
- คะแนนน้อย = dexterity ดี

## 6. Dexterity vs. Strength ในผู้สูงอายุ

| งานวิจัย | ข้อค้นพบ |
|---------|----------|
| Bao et al. (2025) — *Innovation in Aging* | ผู้สูงอายุมี hand dexterity limitation มากกว่า strength limitation (p=0.01) |
| Bao et al. (2025) — Gender effect | เพศหญิง: dexterity limitation เพิ่มตามอายุ (β=0.038, p=0.005) |
| Bao et al. (2025) — Strength | Strength limitation ค่อนข้างคงที่ ไม่ขึ้นกับอายุ |
| *Frontiers in Aging Neuroscience* (2025) | Hand dexterity (NHPT) predict cognition ได้ดีกว่า grip strength |
| *Frontiers in Aging Neuroscience* (2025) | Dexterity + mobility = ~50% variance of cognitive score |

## 7. การประยุกต์กับ AI Vibe Coding

| Clinical Test | Video-based Equivalent |
|--------------|----------------------|
| 9HPT — จับเวลาใส่ peg | MediaPipe — measure speed per trial |
| PPBT — นับจำนวน | Count finger taps / movements |
| FDT — time + penalty | Speed + Accuracy score |
| JHFT — ADL subtest | Task-specific analysis |
| Symmetry index | L vs R comparison |

### Feature Mapping

| มาตรฐานคลินิก | Video Metric | MediaPipe Implementation |
|--------------|-------------|------------------------|
| Completion time | Movement duration | Track first-last hand appearance |
| Error count | Path deviation | Fingertip-to-target distance |
| Smoothness (qualitative) | Jerk/SPARC | 3rd derivative of landmark pos |
| Grip strength | Hand aperture | Distance: thumb tip to finger tips |
| Coordination | Inter-joint correlation | Joint angle covariance |
